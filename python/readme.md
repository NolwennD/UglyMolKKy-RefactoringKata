## Guidelines :

This project uses [uv](https:docs.astral.shuv) for Python project management.

### Prerequisites

Make sure you have 'uv' installed on your system.

### Installing and running the tests

To install the dependencies and run the tests, run the following command from the root of the project:

```bash 
cd python 
uv run pytest
```

`uv` will automatically take care of creating the virtual environment and installing the dependencies listed in the `pyproject.toml` file.
